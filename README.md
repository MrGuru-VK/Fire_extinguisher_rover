This rover is controlled by external remote control car circuit and then act like a toy car but it has a stationary fire extingusher setup on it 
The Required components are 
sno       components             useage 
1         Arduino uno            main microcontroller 
2         flame sensor X3        detecting fire 
3         servo motor            direct the pipe towards the fire 
4         water pump             used to pump the water through the pipe towards fire 
5         motor+wheels X4        for rover
6         toy car controller     for controlling the rover 
7         toy car reciver        for the rover 
This project is mainly to put the use in the small areas where the fire is in any small areas instead of wasting water we can use this
